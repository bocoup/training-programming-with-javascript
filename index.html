Setup Instructions

  1. npm install
  2. npm install -g grunt-cli
  3. Start auto-reloading web-server for your exercise by running grunt.
  4. Open http://localhost:8000 or http://localhost:8000/test in your browser.

Learning Objectives

Values and Variables

  • store values in variables (01)
  • store the return value of a function call in a variable (01)
  • use comparison operators to store booelan values in variables (02)

Arithmetic Operators and the Math Object

  • Use Math.random() to generate a random number (00)
  • use the floor method of the Math Object to round a number (01)
  • use arithmetic operators on variables (01)
  • use the ++ shorthand to increment a value (03)

Strings

  • call the toLowerCase method on String (02)
  • call the indexOf< /a> method on a String (02)
  • use regular expressions to validate user input (02)

Arrays

  • create an empty array object (01)
  • access the property length< /a> on an array (01)
  • access a value in an array using bracket notation and an index (01)
  • add elements to an array using the push function (03)
  • use slice to create a new array by selecting a portion of another array (06)
  • use join to build a string from an array (06)

Control Flow

  • use comparisons to change control flow using if...else statements< /a> (02)
  • iterate over an array using a < code>for loop (03)
  • nest statements and loops to properly execute game logic (03)

Functions

  • Call a function with a parameter (00)
  • return a value from a function (01)
  • define a function that takes an argument (04)
  • define a function that returns a value (04)
  • refactor a function into multiple functions to improve flexibility (04)

Browser and DOM Manipulation

  • Select an element in the DOM with the getElementById function (00)
  • Add content to the DOM using the innerHTML function (00)
  • use alert to deliver information to the user in the browser (05)
  • use prompt to receive user input in the browser (05)
